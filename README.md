# Clasificador de Células Cancerígenas mediante Machine Learning

## Descripción del proyecto

La detección temprana de células cancerígenas es un problema fundamental en el diagnóstico médico. El análisis manual de imágenes microscópicas puede ser un proceso lento y propenso a errores humanos, especialmente cuando se trabaja con grandes volúmenes de datos biológicos.

Este proyecto presenta el desarrollo de un **modelo de clasificación utilizando Machine Learning** para identificar células potencialmente cancerígenas a partir de imágenes microscópicas.

El modelo fue entrenado utilizando un **dataset público disponible en Kaggle**, que contiene imágenes celulares previamente etiquetadas.

El objetivo del proyecto es construir un sistema capaz de **predecir la probabilidad de que una célula sea cancerígena** a partir de una imagen proporcionada.

---

## Problema

El análisis de imágenes celulares presenta múltiples desafíos:

- Gran variabilidad en la morfología celular
- Alto volumen de datos biológicos
- Necesidad de herramientas computacionales que ayuden a los especialistas

Mediante técnicas de **Machine Learning**, es posible entrenar modelos capaces de identificar patrones en las imágenes y ayudar en la clasificación automática de células.

---

## Dataset

El modelo fue entrenado utilizando un **dataset público obtenido de Kaggle**, compuesto por imágenes microscópicas de células.

Cada imagen se encuentra etiquetada indicando si corresponde a:

- Célula cancerígena
- Célula no cancerígena

Estas etiquetas permiten entrenar el modelo para reconocer patrones asociados a células cancerígenas.

---

## Metodología

El proyecto se desarrolló siguiendo las etapas típicas de un pipeline de Machine Learning:

1. **Carga y preparación de datos**
2. **Análisis exploratorio del dataset**
3. **Entrenamiento del modelo de clasificación**
4. **Evaluación del modelo**
5. **Predicción sobre nuevas imágenes**

El modelo aprende a partir de datos previamente etiquetados y luego puede realizar predicciones sobre nuevas imágenes ingresadas por el usuario.
